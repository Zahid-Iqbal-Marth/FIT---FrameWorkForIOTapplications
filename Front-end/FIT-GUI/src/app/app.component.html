FIT : FrameworkforIoT Applications

FIT is a framework that allows users to create complex IoT applications that involve stream processing, event processing, and complex event processing. FIT is a generic framework aimed to be usable by the diverse users of CEP. A GUI is built upon the framework for generic as well as specific applications. Our GUI will enable users to add components to create the application's pipeline.
